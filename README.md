# md-auto-translator

## 概要
Markdown ファイルを指定した言語に翻訳するための、Golang製の個人用ツールです。

ChatGPT を活用して、元の Markdown 構造をできるだけ保ちながら、本文を指定言語へ自動翻訳することを目的として作成しました。
個人利用を前提とした小規模なツールですが、Golang を用いたファイル処理や外部API連携の練習も兼ねています。

## 使用技術
- Go
- ChatGPT API
- Markdown ファイル処理
- 設定ファイルによる翻訳条件管理

## 主な機能
- Markdown ファイルの読み込み
- 指定言語への自動翻訳
- Markdown の構造をなるべく維持したまま本文を変換
- 翻訳結果の出力ファイル生成
- 設定ファイルを用いた翻訳対象・出力内容の管理

## 想定用途
- Markdown ドキュメントの多言語化
- 個人メモや記事下書きの翻訳補助
- AI API を利用した翻訳処理の検証
- Golang によるツール開発の学習

## ファイル構成
- main.go : メイン処理
- go.mod : 依存関係管理
- config/ : 設定ファイル
- original/ : 元の Markdown ファイル
- result/ : 翻訳後の出力ファイル
- README.md : プロジェクト説明

## 実行方法
1. Go をインストール
2. 必要な設定を行う（token）
3. 翻訳対象の Markdown ファイルを用意
4. 実行

go run main.go

※ 実行前に API 利用に必要な設定値や認証情報をご自身の環境に合わせて設定してください。

## 補足
- 個人利用を前提とした小規模なツールです
- 実務案件ではなく、個人開発として作成したものです
- AI 翻訳の精度や出力形式は、入力内容や設定条件によって変動する場合があります
